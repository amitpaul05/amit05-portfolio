Today I spent time fixing a few issues by tracing them properly instead of just patching.

An avatar upload was failing with a 500 error. Traced it down to middleware where multer was getting an empty config due to a disabled default endpoint in production. Chat uploads were unaffected because they use a different path. Fixed the config and verified behavior.

Then I worked on a login issue where users saw a generic failure message. Traced the flow from Keycloak to backend and frontend and found the email was getting dropped during error handling. Updated the flow so the email is preserved and shown in the UI.

Also removed some unused logic in a Django serializer after confirming it’s no longer used in the current flow, and ran tests to ensure nothing broke.

Finally fixed a file upload failure caused by nginx request size limit and improved the frontend message for clarity.

Mostly a day of tracing issues across layers and cleaning up small gaps.
