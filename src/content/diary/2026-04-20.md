Spent 3 hours today just on graphs. No new problems — pure pattern drilling.

**What I drilled:**
- BFS for shortest path (unweighted). The queue + visited-set pattern is now muscle memory.
- DFS for connected components and cycle detection. Recursive vs iterative — I prefer iterative for large inputs to avoid stack overflow.
- Topological sort (Kahn's algorithm). This one kept tripping me up. Finally understood: in-degree array + zero-in-degree queue = clean topo order.

**Problems solved:**
- #207 Course Schedule — first attempt AC using Kahn's. The in-degree reduction is elegant.
- #133 Clone Graph — BFS with a node map. Straightforward once you stop overthinking "deep copy".
- #417 Pacific Atlantic Water Flow — multi-source BFS from both oceans simultaneously. Took two attempts.

**Lesson:** I was solving graph problems without a mental template. Now I have one. BFS = level-by-level, use when distance matters. DFS = exhaustive explore, use when you need all paths or components.
